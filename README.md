# request-loger
> color coded http request loger

## output
* METHOD
* URL
* HTTP VERSION
* HEADERS
* JSON body on put/post/patch/delete
 * *error:* If body is not valid json

## install
`npm install request-loger`

## run
``` sh 
# start on default port 7070
requset-loger
```
``` sh 
# start on default port 4000
PORT=4000 requset-loger
```
